# Tables
I hate tables in HTML but still I gotta know how it works:
```html
<table>
        <thead>
            <tr>
                <th>Number</th>
                <th>Pokemon</th>
                <th>Element</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>25</td>
                <td>Pikachu</td>
                <td>Electric</td>
            </tr>
            <tr>
                <td>4</td>
                <td>Charmander</td>
                <td>Fire</td>
            </tr>
        </tbody>
</table>
```
